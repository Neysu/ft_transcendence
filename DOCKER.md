# ft_transcendence

before being able to start docker you need to add your user to the group "docker"

sudo usermod -aG docker $USER

and restart your computer

if you don't want to do that you can just add sudo before the command 

to start docker just run 

```bash

docker-compose up --build 

```

to acces the website just go to ft_trans.localhost, localhost or the machine ip
